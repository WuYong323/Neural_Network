# 两层神经网络反向传播：公式推导与理解

> 目标：不只是"记住公式"，而是**每一行都能推导出来**，知道它从哪里来。

## 一、全局视角：形状与符号约定

| 符号 | 含义 | 形状 |
|------|------|------|
| X | 输入数据 | (N, 784) |
| W1, b1 | 第一层参数 | (784, 256), (256,) |
| W2, b2 | 第二层参数 | (256, 10), (10,) |
| Y | 真实标签（one-hot） | (N, 10) |
| N | batch 样本数 | 标量 |

**约定**：本文所有 `dX` 表示 ∂L/∂X 乘以 N（即不含 1/N 的"原始"梯度），仅在计算参数梯度时除以 N。这是常见的实现约定，后文会专门说明为何这样做是对的。

---

## 二、前向传播（复习，建立推导基础）

```
Z1 = X @ W1 + b1        # (N, 784) @ (784, 256) → (N, 256)
A1 = ReLU(Z1)           # (N, 256)，逐元素：max(0, x)
Z2 = A1 @ W2 + b2       # (N, 256) @ (256, 10)  → (N, 10)
A2 = softmax(Z2)        # (N, 10)，逐行归一化
L  = -mean(∑ Y * log(A2), axis=1)   # 标量，交叉熵 Loss
```

展开 Loss（对单样本 n，标签类别为 c）：

```
L_n = -∑_k  Y_nk * log(A2_nk)
    = -log(A2_nc)          # 因为 Y 是 one-hot，只有真实类别项非零
L  = (1/N) * ∑_n L_n       # 对 batch 取均值
```

---

## 三、反向传播总览

反向传播的本质是**链式法则**。我们要求 ∂L/∂W1, ∂L/∂b1, ∂L/∂W2, ∂L/∂b2。

依赖链如下（箭头方向 = 反向传播方向）：

```
L ← A2 ← Z2 ← W2, b2
              ↑
              A1 ← Z1 ← W1, b1
                         ↑
                         X
```

按此链条，逐步从后往前推导。

## 四、公式 1：dZ2 = A2 - Y（最关键公式）

### 4.1 为什么这个公式最重要？

Softmax 的梯度单独求很复杂（需要 Jacobian 矩阵），但与交叉熵 Loss **合并后奇迹般地化简**。理解这个推导，是掌握分类网络的核心。

### 4.2 推导过程（单样本，以下先忽略 1/N）

**Step 1：Softmax 的定义**

对第 n 个样本的第 k 个输出：

$$A2_{nk} = \frac{e^{Z2_{nk}}}{\sum_j e^{Z2_{nj}}}$$

**Step 2：对 Loss 求关于 Z2_ni 的偏导**

固定样本 n，求 ∂L_n/∂Z2_ni：

$$\frac{\partial L_n}{\partial Z2_{ni}} = -\sum_k Y_{nk} \cdot \frac{\partial \log A2_{nk}}{\partial Z2_{ni}}$$

利用 ∂log(f)/∂x = (1/f) * ∂f/∂x，展开 Softmax 的 Jacobian：

$$\frac{\partial A2_{nk}}{\partial Z2_{ni}} = \begin{cases} A2_{ni}(1 - A2_{ni}) & k = i \\ -A2_{nk} \cdot A2_{ni} & k \neq i \end{cases}$$

**Step 3：代入求和，分 k=i 和 k≠i 两部分**

$\frac{\partial L_n}{\partial Z2_{ni}}
= -Y_{ni} \cdot \frac{1}{A2_{ni}} \cdot A2_{ni}(1-A2_{ni})+\sum_{k \neq i} Y_{nk} \cdot \frac{1}{A2_{nk}} \cdot (-A2_{nk} \cdot A2_{ni})$

$$= -Y_{ni}(1 - A2_{ni}) + A2_{ni} \sum_{k \neq i} Y_{nk}$$

$$= -Y_{ni} + Y_{ni} \cdot A2_{ni} + A2_{ni} \sum_{k \neq i} Y_{nk}$$

$$= -Y_{ni} + A2_{ni} \underbrace{\left(Y_{ni} + \sum_{k \neq i} Y_{nk}\right)}_{= \sum_k Y_{nk}}$$

**Step 4：利用 one-hot 性质**

Y 是 one-hot 编码，因此 $\sum_k Y_{nk} = 1$，代入得：

$$\boxed{\frac{\partial L_n}{\partial Z2_{ni}} = A2_{ni} - Y_{ni}}$$

**Step 5：加上 1/N（batch 均值）**

$$\frac{\partial L}{\partial Z2} = \frac{A2 - Y}{N}$$

### 4.3 实现时的约定

代码中写 `dZ2 = A2 - Y`（不除以 N），是把 1/N 推迟到计算 dW2 时才除。因为梯度是线性传播的，这完全等价：

```
# 两种写法完全等价：
# 写法 A（数学严格）：
dZ2 = (A2 - Y) / N
dW2 = A1.T @ dZ2         # 已含 1/N

# 写法 B（本文约定）：
dZ2 = A2 - Y             # 不含 1/N
dW2 = A1.T @ dZ2 / N    # 在这里除以 N
```

## 五、公式 2 & 3：dW2 和 db2

### 推导思路

Z2 = A1 @ W2 + b2，即 $Z2_{nk} = \sum_j A1_{nj} \cdot W2_{jk} + b2_k$

**∂L/∂W2_jk：**

$$\frac{\partial L}{\partial W2_{jk}} = \sum_n \frac{\partial L}{\partial Z2_{nk}} \cdot \frac{\partial Z2_{nk}}{\partial W2_{jk}} = \sum_n \frac{1}{N}(A2_{nk} - Y_{nk}) \cdot A1_{nj}$$

写成矩阵乘法（所有 j, k 同时计算）：

$$\frac{\partial L}{\partial W2} = A1^T \cdot \frac{A2 - Y}{N}$$

用约定符号：

```
dW2 = A1.T @ dZ2 / N    # (256, N) @ (N, 10) → (256, 10) ✓
```

**∂L/∂b2_k：**

$$\frac{\partial L}{\partial b2_k} = \sum_n \frac{1}{N}(A2_{nk} - Y_{nk}) \cdot 1$$

```
db2 = sum(dZ2, axis=0) / N    # 对 batch 求和再除以 N → (10,) ✓
```

> **直觉**：b2 对每个样本的 Z2 贡献相同（+1），所以梯度就是对所有样本的 dZ2 求和。

---

## 六、公式 4：dA1（梯度穿过 W2 矩阵）

### 推导思路

Z2 = A1 @ W2，即 $Z2_{nk} = \sum_j A1_{nj} \cdot W2_{jk}$

**∂L/∂A1_nj：**

$$\frac{\partial L}{\partial A1_{nj}} = \sum_k \frac{\partial L}{\partial Z2_{nk}} \cdot \frac{\partial Z2_{nk}}{\partial A1_{nj}} = \sum_k \frac{1}{N}(A2_{nk} - Y_{nk}) \cdot W2_{jk}$$

写成矩阵乘法：

$$\frac{\partial L}{\partial A1} = \frac{A2 - Y}{N} \cdot W2^T$$

用约定符号（dA1 不含 1/N）：

```
dA1 = dZ2 @ W2.T    # (N, 10) @ (10, 256) → (N, 256) ✓
```

> **直觉**：误差从 Z2 "反向流过" W2 矩阵，就像前向传播时 A1 通过 W2 传到 Z2 一样，只是转置。

---

## 七、公式 5：dZ1（ReLU 梯度）

### ReLU 的导数

$$\text{ReLU}(x) = \max(0, x)$$

$$\frac{d \text{ReLU}(x)}{dx} = \begin{cases} 1 & x > 0 \\ 0 & x \leq 0 \end{cases}$$

### 应用链式法则

A1 = ReLU(Z1)，所以：

$$\frac{\partial L}{\partial Z1_{nj}} = \frac{\partial L}{\partial A1_{nj}} \cdot \frac{\partial A1_{nj}}{\partial Z1_{nj}} = \frac{\partial L}{\partial A1_{nj}} \cdot \mathbb{1}[Z1_{nj} > 0]$$

```
dZ1 = dA1 * (Z1 > 0)    # 逐元素相乘，(N, 256) ✓
```

> **直觉**：ReLU 在正数区域梯度为 1（直接传递），在负数区域梯度为 0（**截断梯度**，不往前传）。这就是"死亡 ReLU"现象的根源——某个神经元如果始终在负数区，它的梯度永远是 0，参数永远不更新。

### 注意：为什么用原始的 Z1，而不是 A1？

`Z1 > 0` 是在判断激活**之前**的值是否为正。用 A1 判断不对，因为 A1 = ReLU(Z1) ≥ 0 始终非负，会错误地将 Z1=0 的位置也传递梯度。

---

## 八、公式 6 & 7：dW1 和 db1

### 推导（与 dW2、db2 完全对称）

Z1 = X @ W1 + b1，即 $Z1_{nj} = \sum_i X_{ni} \cdot W1_{ij} + b1_j$

**∂L/∂W1_ij：**

$$\frac{\partial L}{\partial W1_{ij}} = \sum_n \frac{\partial L}{\partial Z1_{nj}} \cdot X_{ni}$$

```
dW1 = X.T @ dZ1 / N     # (784, N) @ (N, 256) → (784, 256) ✓
db1 = sum(dZ1, axis=0) / N   # (256,) ✓
```

---

## 九、参数更新

```python
W2 -= lr * dW2    # (256, 10)
b2 -= lr * db2    # (10,)
W1 -= lr * dW1    # (784, 256)
b1 -= lr * db1    # (256,)
```

梯度下降的含义：沿 Loss 下降最快的方向移动一小步（步长 = lr）。

## 十、完整代码实现

```python
import numpy as np

def softmax(Z):
    # 减去最大值防止数值溢出（exp(大数) → inf）
    expZ = np.exp(Z - Z.max(axis=1, keepdims=True))
    return expZ / expZ.sum(axis=1, keepdims=True)

def forward(X, W1, b1, W2, b2):
    Z1 = X @ W1 + b1           # (N, 256)
    A1 = np.maximum(0, Z1)     # ReLU
    Z2 = A1 @ W2 + b2          # (N, 10)
    A2 = softmax(Z2)           # (N, 10)
    return Z1, A1, Z2, A2

def cross_entropy_loss(A2, Y):
    N = Y.shape[0]
    return -np.mean(np.sum(Y * np.log(A2 + 1e-8), axis=1))

def backward(X, Y, Z1, A1, A2, W2):
    N = X.shape[0]
    dZ2 = A2 - Y               # (N, 10)
    dW2 = A1.T @ dZ2 / N      # (256, 10)
    db2 = dZ2.sum(axis=0) / N  # (10,)
    dA1 = dZ2 @ W2.T           # (N, 256)
    dZ1 = dA1 * (Z1 > 0)      # (N, 256)
    dW1 = X.T @ dZ1 / N       # (784, 256)
    db1 = dZ1.sum(axis=0) / N  # (256,)
    return dW1, db1, dW2, db2

def update(W1, b1, W2, b2, dW1, db1, dW2, db2, lr):
    W1 -= lr * dW1
    b1 -= lr * db1
    W2 -= lr * dW2
    b2 -= lr * db2
    return W1, b1, W2, b2
```

## 十一、公式一览表

| 公式 | 形状 | 本质 |
|------|------|------|
| `dZ2 = A2 - Y` | (N, 10) | softmax + 交叉熵合并梯度 |
| `dW2 = A1.T @ dZ2 / N` | (256, 10) | X^T · δ（矩阵乘法求权重梯度） |
| `db2 = sum(dZ2, axis=0) / N` | (10,) | 对 batch 求和（偏置共享） |
| `dA1 = dZ2 @ W2.T` | (N, 256) | δ · W^T（梯度穿过线性层） |
| `dZ1 = dA1 * (Z1 > 0)` | (N, 256) | 逐元素门控（ReLU 导数） |
| `dW1 = X.T @ dZ1 / N` | (784, 256) | 同 dW2，对称结构 |
| `db1 = sum(dZ1, axis=0) / N` | (256,) | 同 db2 |

## 十二、常见错误与陷阱

### 1. 忘记 softmax 减去最大值

```python
# 错误（大 Z2 会产生 inf）
np.exp(Z2) / np.exp(Z2).sum(axis=1, keepdims=True)

# 正确
expZ = np.exp(Z2 - Z2.max(axis=1, keepdims=True))
expZ / expZ.sum(axis=1, keepdims=True)
```

### 2. dZ1 用错了 A1 而不是 Z1

```python
dZ1 = dA1 * (A1 > 0)   # 错！A1 = ReLU(Z1) >= 0，条件始终成立
dZ1 = dA1 * (Z1 > 0)   # 正确
```

### 3. sum 忘写 axis=0

```python
db2 = np.sum(dZ2) / N        # 错！把所有元素加在一起了
db2 = np.sum(dZ2, axis=0) / N  # 正确：对 N 个样本求和，保留 10 维
```

### 4. log 里没有加数值稳定项

```python
loss = -np.mean(np.sum(Y * np.log(A2), axis=1))        # 危险！A2=0 时 log 为 -inf
loss = -np.mean(np.sum(Y * np.log(A2 + 1e-8), axis=1)) # 安全
```

## 十三、一句话总结每条公式的逻辑

```
dZ2 = A2 - Y         → "预测值与真实值的差"，是损失对 Z2 的直接信号
dW2 = A1.T @ dZ2 / N → "输入（A1）的转置 × 输出误差"
db2 = Σ dZ2 / N      → "偏置影响每一个样本，梯度叠加再平均"
dA1 = dZ2 @ W2.T     → "误差反向穿过 W2，传回 A1"
dZ1 = dA1 * (Z1>0)  → "ReLU 正区通，负区断"
dW1 = X.T @ dZ1 / N  → 与 dW2 完全对称
db1 = Σ dZ1 / N      → 与 db2 完全对称
```
